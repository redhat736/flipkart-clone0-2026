# Flipkart Clone 2026

A static Flipkart-like e-commerce website clone built with HTML, CSS, and JavaScript.

## Project Structure

- `index.html` — Main homepage
- `login.html`, `register.html`, `profile.html`, `cart.html`, `payment.html`, etc.
- `css/` — Styling files
- `js/` — Front-end JavaScript logic
- `images/` — Image assets

## Features

- Static product pages and category pages
- Shopping cart interactions
- User login/register UI flow
- Responsive navigation bar
- Search page and contact page

## Run Locally

Open `index.html` in your browser, or serve the folder with a local HTTP server for best results.

### Using Python (if available)

```bash
cd e:\flipkart-clone-2026
python -m http.server 5500
```

Then open `http://localhost:5500` in your browser.
